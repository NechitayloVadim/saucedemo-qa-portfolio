# saucedemo-qa-portfolio
Manual QA testing project for SauceDemo web application
## Описание проекта:
Ручное тестирование веб-приложения SauceDemo.
## Покрытие тестирования:
- Функциональное тестирование
- Позитивные сценарии
- Негативные сцинарии
- Проверка обязательных полей
- Проверка валидации данных
- UI проверки
## Артефакты тестирования:
- 32 тест-кейса
- 3 чек-листа
- 1 баг-репорт
## Окружение:
- Windows 10
- Yandex Browser
- https://www.saucedemo.com/
## Цель проекта:
Продемонстрировать навыки ручного тестирования, составления тестовой документации и поиска дефектов.
